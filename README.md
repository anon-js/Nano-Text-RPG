Nano Bot (KakaoTalk Bot) Game Module

메신저봇R 기준으로 돌아가는 소스입니다.

## 패치 내역
Ver 0.0.1
> Bridge 사용하지 않고 module로 변경.

## 구현 된 기능

```js
.join(); //가입
.info(); //내정보
.remove(); //탈퇴
.move(/*장소*/); //이동
.read_item(); //아이템
.live(); //부활
```

## 구현 중인 기능

```js
.training(); //훈련
.use_item(/*이름*/, /*갯수*/); //아이템 사용
```
