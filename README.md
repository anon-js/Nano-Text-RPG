## 구현 된 기능

```js
.join(); // 가입

.info(); // 내정보

.remove(); // 탈퇴

/**
 * 필요한 값
 * place_name: 장소 이름
 */
.move(place_name); // 이동

.read_item(); // 아이템

.live(); // 부활

.training(); // 훈련

/**
 * 필요한 값
 * item_name: 아이템 이름
 * item_int: 아이템 갯수
 */
.use_item(item_name, item_int); // 아이템 사용
```

## 구현 중인 기능
