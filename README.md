## 구현 된 기능

```js
.join(); // 가입

.info(); // 내정보

.remove(); // 탈퇴

.move(place_name); // 이동
/**
 * place_name: 장소 이름
 */

.read_item(); // 아이템

.live(); // 부활

.training(); // 훈련

.store(); // 상점

.map(); // 지도

.use_item(item_name, item_int); // 아이템 사용
/**
 * item_name: 아이템 이름
 * item_int: 아이템 갯수
 */
```

## 구현 중인 기능

```js
buy(); // 구매

battle(); // 전투
```

## 예시 ex)

index.js
```js
const rpgGame = require("Rpg.js"); // 모듈 불러오기 (예시)
const rpg = rpgGame(path, va, fs);
/**
 * path: 파일 경로
 * va: 전체보기 ['\u200b'.repeat(500);]
 * fs: FileStream
 */
```
```js
rpg.init(s, r); // 함수 불러오기 (예시)
/**
 * s: sender
 * r: replier
 */
rpg.함수명(/* 필요 시 매개변수 */)
```


사용자가 기본 지식을 가지고 있지 않으면 사용하는데 어려움이 있을 수 있습니다.
